Download Theme Package

After purchasing successfully our theme from Themeforest, you will need to download theme package to use. To download the package, please login to your Themeforest account, navigate to “Downloads”, click the Download button in the left of the theme, choose All files & documentation to download our theme package to your computer. The theme package contains:

  • Wilcity Theme File: Includes wilcity.zip file that consists of everything you need to install the theme.
  • Document Folder: Includes Wilcity Documentation file that helps you get to know about Wilcity and guides you to build your site with our theme.
  • Licensing Folder: Includes the licensing files of Themeforest purchase.

Requirement For Wilcity

To use Wilcity theme, You must be running WordPress 4.0 or higher, PHP 5.3 or higher, Mysql 5 or higher.

Installation

Installing Theme

Once you have downloaded the Wilcity package from Themeforest.net, you can choose one of the two methods to install this theme.

  • Installation Via FTP

    Extract the wilcity.zip file and upload the extracted folder to the /wp-content/themes/ folder on your server.

  • WordPress

    Navigate to Apperance -> Add New Themes -> Upload page. Select the wilcity.zip file.

    Press the Install Now button to upload and install the theme.

    After uploading the theme, you must activate it. Navigate to the Appearence -> Themes page to activate the theme.

    Notice: If you get this error “HTTP Error 413 Request entity too large” or an error relate to Solve the Exceeds the Maximum Upload Error, please try extracting the download package -> you will see the wilcity folder in there -> Referring “Installation via FTP” way.

Installing Plugins

After you have actived Wilcity, you will see a notice at the top of the screen. This notice warns you about 8 plugins that are required and recommeded to be installed along with the theme (or click on Appearance->Install plugin). Click on Begin installing plugins to install/active the plugins one by one.

Importing Demo

  • In the package downloaded, you will see a folder called dummy data, this folder contains all import files
  • To import the dummy data, please log into your website -> Tools -> Import

Updating Theme

There are 2 methods to update the theme:

  • Manually via WordPress admin: After downloading the package from Themeforest, you can delete the old theme and install the new theme. All your settings with our theme will be preserved but it sometimes results in losing Menu or Widgets settings because this method actually switches on/off the menu and widget location which has been registered by this theme.
  • Manually via FTP:After downloading the package from Themeforest, you can extract it and using FTP client like FileZilla to remove the old theme and upload the new theme. With this method, all your settings, including the menu and widget will be persevered.

We highly recommmend following https://facebook.com/wilokewp to get a notification of each update.

Setup Homepage

The theme supported King Composer plugin, so you can create a lot of styles on Homepage by using drag & drop functionality. Following these steps below to create a cool homepage for your website:

  • Click on Add New sub menu of the Pages menu
  • Enter a title for this page.
  • Switch WordPress Editor to Backend Editor mode by clicking Backend Editor button.
  • Click Add Element, a popup of shortcodes will be shown now
  • Here you can choose shortcodes that you want to use for the page.
  • We need to set the page as Visual Page before click Publish button. In the Page Attributes box, select Visual Page for Template setting.
  • Now click Publish button
  • Finally, go to Settings->Reading->Checked A Static page and assign Front page to Homepage (It's page title that was created previous step)->Click Save Changes
